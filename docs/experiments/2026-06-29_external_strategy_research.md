## 二、2026 年量化策略趋势（8个方向）

1. **Value + ML（非线性因子融合）** — LightGBM/CatBoost 替代线性加权
   - BigQuant 已有验证：36.03%年化，回撤仅 7.51%
   - 我们 4 次 ML 证伪（v30/v47/ml_hybrid80/权重预测），根因是截面信号太弱
   - **但 BigQuant 的成功说明 ML 不是完全不可用，而是需要正确的因子输入**

2. **情绪+成交量结构** — 主力净流入 + 大单占比 + 换手率
   - 我们已有资金流因子（v50 证伪，但数据仅120天）
   - 需要更长期的资金流数据

3. **因子宏观条件** — 动态调整因子权重
   - 我们 WF 不稳定的根因之一就是因子权重固定
   - 需要设计宏观状态机（3-4个状态）

4. **ETF轮动+风险平价** — 跨资产配置
   - 适合大资金、低风险偏好
   - 我们目前只做个股，这是新方向

5. **Execution-aware 下单** — 滑点/冲击模型
   - 我们当前成交价模型太简单（固定滑点0.01%）
   - 需要按成交量分布、持仓量估算冲击成本

6. **高频信号中低频融合** — 分钟+日频混合
   - 需要分钟级行情数据（我们没有）

7. **因果/可解释性（SHAP）** — 剔除不稳定特征
   - AlphaPurify 的 FactorAnalyzer 可以对接

8. **主题/事件驱动** — 新闻情绪+另类数据
   - v48 已尝试（业绩预告/增减持），事件因子 IC ≈ 0
   - 另类数据（卫星/供应链）获取成本高

## 三、短线策略设计方案（6个）

### 3.1 窄震出趋势（可落地 ✅）
- **逻辑**: 波动率压缩 → 放量突破 → 持有 5-10 日
- **信号**: 20日振幅<15% + 量比>2 + 突破箱体上沿及20日均线
- **风控**: 跌回箱体止损，浮盈15%分批止盈
- **数据需求**: 纯量价，现有数据 100% 够用

### 3.2 超跌+资金承接（可落地 ✅）
- **逻辑**: 跌幅深 + 主力净流入转正 + 放量止跌 → 2-5日反弹
- **信号**: 5-10日跌幅>指数 + RSI超卖 + 资金流转正 + 放量长下影
- **风控**: 继续创新低止损，3-5日无起色离场
- **数据需求**: 量价 + 资金流（已有）

### 3.3 高质量成长波段（待数据 ⚠️）
- **/增速好的公司 + 技术节奏做波段
- **数据需求**: PE/PB/ROE/净利润增速（我们没有）

### 3.4 日内波动捕捉（待数据 ⚠️）
- **逻辑**: 盘内区间反转/突破（早盘假跌真拉/午后放量突破）
- **数据需求**: 分钟级行情（我们没有）

### 3.5 主题轮动地图（待数据 ⚠️）
- **逻辑**: 板块强度评估 → 板块内强势股 → 3-7日轮动
- **数据需求**: 行业/概念分类（部分有，需补充）

### 3.6 风格因子切换（可落地 ✅）
- **逻辑**: 监控小盘/大盘、成长/价值相对强度 → 做风格补涨
- **数据需求**: 风格指数数据（中证500/1000 vs 沪深300）

## 六、关键结论

1. **简单量价因子日频截面 IC 很弱**（AlphaPurify 验证），需要非线性组合或加入新数据源
2. **线性多因子（PE⁻¹+ROE+动量+市值）** 是 BigQuant 上验证成功的最简单方向，年化 41%
3. **窄震突破 + 超跌反弹** 是 A 股震荡市最匹配的短线策略，且我们数据齐全
4. **ML 截面选股** 我们 4 次证伪，但 BigQuant 的 LightGBM 36%/7.51%回撤说明不是 ML 无用，而是因子输入不对
5. **行业中性化** 是我们缺失的关键环节，AlphaPurify 正好补上
