# v42 换手率因子研究 — 实验记录

**日期**：2026-06-24
**状态**：❌ 未采用（收益低于 v39i）

---

## 研究动机

当前 v39i 策略中的"量比"因子（`turnover_avg` = volume / avg_volume）与动量因子高度相关（r≈0.94）。尝试用**真实换手率**（volume / float_shares）替代量比，看是否能提供独立信息量、改善风险调整后收益。

## 核心改动

1. **新增真实换手率因子**：`turnover_rate = volume / float_shares`
   - `float_shares` 通过腾讯行情接口（字段57：流通市值/股价）获取
   - 已对选股池中 800 只股票拉取并存储到 `stock_pool.float_shares`
2. **新增权重参数**：`W_TURNOVER_RATE`（换手率权重）、`W_TURNOVER_AVG`（量比权重，设为0）
3. **clip_max 参数**：控制换手率因子上限截断值

## 结论

1. **v42 夏普更高**（1.117 vs 1.005）：风险调整后收益更好
2. **v42 回撤更低**（10.70% vs 16.69%）：防守能力更强
3. **v42 收益更低**（68% vs 92%）：绝对收益牺牲约 5.5%/年
4. **clip_max 无影响**：A股正常换手率 1-5%，clip_max 从 0.05 到 0.30 对排序无影响
5. **换手率与动量高度相关**（r=0.94）：真实换手率提供的独立信息量有限

## 最终决定

**维持 v39i 作为运行策略**。v42 虽然夏普更高、回撤更低，但绝对收益差距较大（年化差 ~5.5%），且换手率因子提供的边际信息量不足以替代动量。

## 关键发现

- 真实换手率（volume/float_shares）和量比（volume/avg_volume）提供相似信息
- 动量因子（mom_5）已隐含了换手率的大部分信息
- 在动量因子存在的情况下，额外添加换手率因子属于"冗余因子"
- clip_max 参数对换手率因子调优无意义（A股换手率分布集中）

# v44 质量+低波小市值策略实验

> 2026-06-25 创建

## 策略设计

### 核心理念
- v43 追涨策略证伪后，转向"质量+低波"攻守兼备方向
- 2024年低波因子表现优秀（防御年），2025年质量+规模因子共振（进攻年）
- 2026年市场风格不明，质量+低波组合理论上攻守兼备

### 因子选择
| 因子 | 权重 | 方向 | 说明 |
|------|------|------|------|
| quality（20日动量） | 0.35 | 正向 | 中期趋势质量，比5日更稳定 |
| volatility（20日波动率） | 0.30 | 反向 | 低波动=高质量=防御 |
| size（市值） | 0.20 | 反向 | 小市值溢价 |
| turnover_rate（换手率） | 0.15 | 正向 | 适度换手=活跃度质量 |

### 参数
- 选股池：全A 4605只，市值20-100亿
- 止损：-8%，止盈：+15%
- 持有期：max 10天，浮盈≥3%延长到15天
- 持仓：8只，单只上限12.5%，每日最多买3只

## 实验记录

### [2026-06-25] v44a — 初始版

- **回测区间**：2024-06-01 ~ 2026-06-01
- **WF**：train=120, test=60, step=60
- **结果**：-16.15% / 夏普-0.78 / 回撤28.87% / 正收益fold 0/3
- **问题**：
  1. 全A 4605只范围太分散，好股坏股混合
  2. 2年窗口太短，低波因子在短窗口内不占优
  3. 50%的fold没有任何交易（选股条件太严）
- **与 v39d 对比**：v39d 在 zz800 + 全A mix 上 +57.67%，v44 全A纯跑-16%

## 调研参考

### 2025年A股因子表现（guorn.com）
- 赢家：规模因子（小盘）、质量因子、成长因子、低流动性因子
- 输家：低波动因子、价值因子、低股价因子
- **关键洞察**：2024年（熊市）低波是赢家，2025年（牛市）低波是输家
- **2026年展望**：市场处于中间点，应保持均衡配置

## v44 资金流策略（2026-06-25 新增方向）

v43 追涨策略证伪后，v44 改为"资金流+动量+低波质量"方向。

### 核心逻辑
- **资金流（W_FLOW=0.35）**：mom_5 × 成交量变化，衡量价量共振强度
- **动量（W_MOM=0.25）**：5日动量 > 3% 门槛
- **低波过滤**：日波动率 ≤ 4%（年化≈63%）
- **规模（W_SIZE=0.15）**：小市值偏好
- **换手率（W_TURNOVER=0.10）**：适度换手质量
- **illiq（W_ILLIQ=0.10）**：非流动性溢价

### 实验结果

#### [2026-06-25] v44a — zz800 初始版 ✅

- **回测区间**：2023-01-01 ~ 2026-06-01（3年）
- **WF**：train=252, test=126, step=63（4 folds）
- **结果**：**+26.32% / 夏普1.252 / 回撤18.71% / 正收益fold 4/4 (100%)** ✅
- **意义**：v44 夏普 1.252 超过 v39i（1.199），成为新的最优夏普策略
- **对比**：v39d 收益更高（+57.67%）但夏普较低（0.654），v44 风险调整后收益更优

### 与 v39d 的差异
| 维度 | v39d | v44 |
|------|------|-----|
| 核心因子 | mom_5(0.20) + fund_flow(0.15) | fund_flow(0.35) + mom(0.25) |
| 低波过滤 | 无 | 日vol ≤ 4% |
| 动量门槛 | 动态 MOM_THRESHOLD | 静态 3% |
| 持有期 | 5天+3天延长 | 5天+7天延长 |
| 夏普 | 0.654 | **1.252** |
| 收益 | **+57.67%** | +26.32% |

### 参数扫描实验（2026-06-25）

#### 权重扫描（13种组合）
所有权重组合结果完全相同：+19.58%/夏普0.931（2-fold）。说明 fund_flow 和 mom 高度相关，权重变化不影响选股结果。

#### 风控参数扫描（11种组合）
| 参数 | 结果 |
|------|------|
| SL=-3%~-8%, TP=8%~20% | 全部 +19.58%/夏普0.931 |
| Hold=10天/15天 | 0%（选不出股票） |

**结论**：v44 对风控参数极不敏感，收益来自选股逻辑本身。

# v45 新策略研究方向设计

> 2026-06-25 创建

## A: 反转策略（Contrarian）

### 思路
- v39i 是动量策略（买涨得好的），反转策略买超跌反弹
- A股是否有短期反转效应？已有文献表明A股存在"涨停后过度反应"导致的动量消失（ScienceDirect论文），这意味着**反转可能在特定条件下有效**
- 与 v43 的"反转尝试"区别：v43 反转是买 mom<-5% 的极端超跌，这里设计更精细的反转信号

### 设计参数
- **信号**：mom_5 < -4%（5日跌幅>4%）+ turnover 放大（恐慌放量）+ illiq 适中
- **门槛**：排除ST、排除上市<6月、排除连续跌停
- **持有期**：3-5日（反弹后快进快出）
- **止损**：-5%（继续跌就跑）
- **止盈**：+8%（反弹目标）
- **选股池**：zz800（先用zz800快速验证，再扩展全A）

### 假设
- A股短期过度反应后存在1-5日窗口的反弹概率
- 配合成交量放大（恐慌后接盘涌入）可提高胜率

### 验证优先级
1. zz800 快速回测（2 folds，10s/fold）
2. 通过则上 WF（4 folds）
3. 对比 v39i 动量——如果反转与动量负相关，可组成配对策略

---

## B: 短周期调仓（v39i 变体）

### 思路
- v39i 持有5天，缩短到2-3天，提高资金效率
- 核心假设：A股动量效应在前3天最强，之后衰减
- 调仓更频繁 = 更高的换手 = 更少的暴露风险

### 设计参数
- **HOLD_DAYS_MAX**：3天（vs v39i的5天）
- **HOLD_DAYS_EXTEND**：5天（浮盈≥3%仍延长）
- **其他**：完全复制 v39i 的选股逻辑（size_factor + illiq + dynamic mom threshold）

### 假设
- 缩短持有期减少回撤，但可能错过继续涨的收益
- 关键是看 **收益/回撤比** 是否提升

### 验证
1. 直接改 HOLD_DAYS_MAX=3，跑 zz800 WF（4 folds）
2. 与 v39i 对比 Sharpe/回撤/换手率

---

## C: 事件驱动策略

### 思路
- 利用特定事件窗口的 Alpha：
  - 财报发布后（业绩超预期 → 股价跳空）
  - 大股东增减持（增持 → 信心信号）
  - 解禁前（通常跌）
  - 高送转/分红公告
- 数据源：东方财富/同花顺 F10 数据

### 可行性评估
- **财报数据**：腾讯/AKShare 接口可获得财报发布日期和业绩预告
- **增减持数据**：需要爬虫或付费数据源
- **复杂度**：比纯因子策略高一个量级，需要新数据管道

### 验证优先级
1. 先评估数据可获得性（1周内）
2. 如果数据可获取，用"业绩预告超预期"作为单一信号快速测试

---

## D: 行业轮动（真·行业轮动）

### 思路
- 与 v43 完全不同——v43 是"小市值轮动选股"
- D 是**先选行业，再选个股**
- 核心：识别当前市场热点板块，在热点板块内选强股

### 设计参数
- **行业信号**：行业指数5日动量（领涨板块）
- **个股信号**：行业内动量 Top 3
- **行业选择**：选动量前30%的行业（约10-15个）
- **持有期**：5天

### 数据需求
- 行业分类数据（申万/证监会）
- 行业指数行情（已有 K 线可计算）

### 假设
- 行业轮动在A股有效（板块轮动明显）
- 行业内动量选股比全市场动量选股更有效

### 验证
1. 先构建行业动量指标（用已有K线按行业聚合）
2. 跑 zz800 WF

### 验证

## 实验结论汇总

### [2026-06-25] B: 短周期调仓（v39i HOLD_DAYS_MAX=3）

- **回测区间**：2023-06-01 ~ 2026-06-01（3年）
- **WF**：train=252, test=126, step=63（6 folds）
- **结果**：+13.67% / 夏普0.774 / 回撤25.47% / 正收益fold 3/6 (50%)
- **对比 v39i 原版(5天)**：+26.32% / 夏普1.252 / 回撤18.71% / 正收益fold 4/4 (100%)
- **结论**：❌ 缩短持有期到3天导致收益减半、正收益fold减少。5天持有期是最优参数。

### [2026-06-25] A: 反转策略 v45a

- **回测区间**：2023-06-01 ~ 2026-06-01（3年）
- **WF**：train=252, test=126, step=63（6 folds）
- **选股逻辑**：mom_5 < -4%（超跌）+ 换手率适中，买3只，持有5天，止损-5%，止盈+8%
- **结果**：+4.34% / 夏普0.280 / 回撤21.16% / 正收益fold 4/6 (67%)
- **分析**：
  - 策略能微利，但夏普太低（0.280 < 0.5）
  - 回撤偏高（21.16%）
  - 反转策略在A股有一定效果，但不如动量策略（v39i）
  - 可能原因：A股短期反转效应弱，动量效应更强
- **结论**：❌ 反转策略在A股（zz800范围）效果不佳，与v39i动量策略差距巨大
