> 评审时间：2026-06-11

## 一、P0 Bug 验证结果

### Bug 1：WF 拼接方式错误（run_backtest.py:580-582）

**报告说**：几何平均每段收益，忽略了累计回撤，导致 v11b WF 年化 63.7% 虚高。

**验证结论**：⚠️ **部分成立，但影响有限**

- fold_results 中每个 fold 的指标（年化/夏普/回撤）是**独立计算**的，不依赖拼接
- combined_nav 仅用于 save_results() 中的展示（Total OOS 拼接统计行）
- 报告中 v11b WF 63.7%/1.70/69% 这些数字来自 fold 独立统计，不是拼接值
- 但拼接的"Total OOS"展示值确实有误导性

**是否需要修**：应该改，但改动影响的是展示值，不影响核心结论

### Bug 2：止损不收印花税（account.py:186）

**报告说**：止损免印花税，违反 A 股规则。

**验证结论**：**完全成立，立即修复** ✅

```python
# 当前代码（错误）
stamp_tax = revenue * costs.stamp_tax_rate if reason != 'STOP_LOSS' else 0.0

# 应改为
stamp_tax = revenue * costs.stamp_tax_rate
```

影响：所有止损卖出少算 0.1% 印花税。对高换手策略影响更大。

### Bug 3：分级止盈跨 tier 漏单（account.py:336-337）

**报告说**：profit 回落导致高 tier 永远不触发。

**验证结论**：**理论上成立，实际影响较小**

- v11b 持有期 20 天，日内 profit 跨 tier 回落的概率不高
- 但确实是正确性 bug

**修复方案**：记录 highest_profit_seen

## 二、改进建议评估

### Regime Switching（市场状态切换）

**报告建议**：三态（牛市/熊市/震荡）+ 不同策略参数

**评估**：❌ **不采纳（作为框架），原因如下**

1. A 股 regime switching 的最大难题是**识别滞后**，不是框架
2. 报告给的阈值（ma60 > 5%, vol60 < 30%）过于简单，样本外效果存疑
3. 我们已经有 Ensemble（3组因子自适应），本质就是隐式 regime switching
4. 显式 regime switching 引入更多参数，WF 更容易过拟合
5. 报告自己也说"v10c_crash3 WF 年化 12.3% → 26.8%"，但 crash3 只是去掉跌幅最大 3 只，不是 regime switching

**替代方案**：如果要做，用 HMM 识别 market regime，而不是阈值

### 风险平价替代等权

**报告建议**：risk_parity_weights() 替代 1/N

**评估**：⚠️ **有价值但优先级不高**

1. 对 8-15 只持仓，风险平价 vs 等权差异不大
2. 我们已经有 vol_scaling（波动率缩放），部分实现了类似效果
3. Sharpe 提升 0.1-0.2 是业内普遍结论，但取决于持仓相关性
4. 实现成本低（2 小时），可以加选项

### 多周期融合

**报告建议**：70% 月度因子 + 30% 周度因子

**评估**：❌ **不采纳**

1. 我们 v11b 是 20 日调仓，本身就在月度和周度之间
2. 多周期融合增加复杂度，收益提升不确定
3. 周度因子噪声大，在中证 800 上优势不明显

### 基本面因子

**报告建议**：加入 EP/BP/ROE/股息率等

**评估**：⚠️ **研究方向正确，但短期不做**

1. 数据源是问题：Tushare 需要 token，且 API 不稳定
2. 我们 40 个技术因子还没做扎实，加基本面是另一个维度的工作
3. Fama-French 五因子归因是**当前最缺的东西**，比加因子更重要

### 北向资金因子

**报告建议**：北向资金净流入作为因子

**评估**：❌ **不采纳**

1. 报告自己说"2018 年后 alpha 已经衰减"
2. 数据获取复杂（需要 Tushare pro）
3. 对中证 800 成分股效果有限

### 事件驱动

**报告建议**：财报超预期/分析师上调/股权激励/回购

**评估**：❌ **短期不做**

1. 需要新的数据源（研报/公告）
2. 实现复杂度高，属于另一个研究方向
3. 现有技术因子框架下优先级不高

### 集成学习

**报告建议**：多周期标签 + stacking

**评估**：⚠️ **长期方向，当前不做**

1. 我们 ml_hybrid80 已经跑过，样本外 IC≈0
2. 报告自己承认"A 股截面噪声极大，IC > 0.03 就算有效"
3. 工程量大，收益不确定

### 股指期货对冲

**报告建议**：多因子 alpha + 股指期货对冲 beta

**评估**：❌ **个人投资者不做**

1. 门槛：50w + 适当性测试
2. 基差风险、展期成本
3. 超出当前项目范围
