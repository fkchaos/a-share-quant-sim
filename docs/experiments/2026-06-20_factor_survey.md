### WF 结果
| 版本 | 正收益fold | 夏普 | 回撤 | 耗时 |

**2024年（防御风格主导）**

- 🥇 **低波动率因子**：年度最佳，风险调整收益最优

- 🥈 **成长因子**：DeepSeek / 人形机器人主题驱动
- 🥉 **质量因子**：ROE / 盈利增长稳定贡献
- 中证流通指数收益率：**24.75%**

> ⚠️ 2024 年"赢家因子"（红利、低波、价值）在 2025 年沦为"输家"——教科书式镜像反转，风格轮动速度超预期。

## 四、AI / 深度学习因子

### 4.1 研究进展

| 方向 | 机构 / 来源 | 核心发现 |
|------|------------|---------|
| LSTM 多因子 | 华福证券 | 80 分析师预期 + 134 资金流 + 43 高频聚合 + 深度学习因子，多空年化 46%（夏普 2.37），多头年化 23% |
| Transformer 选股 | 西南证券 DAFAT | IC 均值从 9.42% 提升至 11.07%，年化 32.30% |
| LSTM + MLP | BigQuant | 中证 1000 周度收益，静态内在属性 + 深度学习年化 8.8%，动态提升至 26.2% |
| DeepSeek 因子优化 | 信达证券 | 75% 的 Alpha158 因子 RankIC 均值提升 |
| AI Agent 因子挖掘 | FinLab | 有记忆 Agent 效率是无记忆 3 倍，IC 达 8.25% |
| Kronos 金融 K 线模型 | 清华 AAAI 2026 | 零样本对 A 股完全无效，需 GPU 微调 |

### 4.2 关键结论

- **深度学习因子在 A 股有效**，但需要足够的数据量和正确的标签设计
- **LSTM / Transformer** 在择时上效果优于选股
- **零样本金融大模型（如 Kronos）对 A 股无效**，必须微调
- **AI 因子与传统因子结合** 效果最优（如 LSTM + 多因子融合）

## 七、核心结论

1. **A 股因子有效性年度差异巨大**：2024 防御 → 2025 进攻，风格反转速度快，需动态配置
2. **量价因子仍是核心**：`high_low_range` / `vol_60` / `pv_corr_20` 长期稳定有效
3. **短期动量在 A 股为反转**：3-6 月回看 IC 为负，残差动量可转正
4. **分析师预期因子被低估**：SUE / 异常覆盖 / 盈利修正 IC 高且与量价因子低相关
5. **日内因子效果惊人**：偏锋涨跌幅 IC = 6.29%，尾盘反转 RankIC = 6.99%
6. **AI 因子有效但需微调**：零样本模型无效，LSTM / Transformer + 多因子融合最优
7. **v27 策略处于正确轨道**：核心因子 `pv_corr_20` 被广泛验证，TP / SL 参数最优
8. **最大增强空间**：分析师预期因子 + 残差动量 + 行业轮动，与现有策略低相关
