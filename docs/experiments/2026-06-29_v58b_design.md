## 1. 核心逻辑

**超跌 → 资金承接 → 2-5日反弹**

不做追涨，专门抄跌幅已深+资金明显回流的股票做短期反弹。和 v39g 互补：
- v39g: 追强（mom>3% + 小市值）
- v58b: 抄底（跌幅深 + 资金回流）

## 2. 信号构造

### 2.1 因子定义

| 因子 | 计算 | 方向 | 含义 |
|------|------|------|------|
| `drop_5d` | close / close.shift(5) - 1 | 越负越好 | |
| `drop_10d` | close / close.shift(10) - 1 | 越负越好 | 10日跌幅 |
| `rsi_14` | RSI(14) | 越低越好(超卖) | 超卖程度 |
| `fund_flow_ratio` | funds_inflow / volume | 越高越好 | 资金流入占比(资金流) |
| `vol_reversal` | volume / volume.shift(1) (on down day) | 越高越好 | 跌势末放量 |
| `recovery_signal` | close > open after N down days | 0/1 | 止跌K线 |

### 2.2 选股条件（硬筛选）

```
drop_10d < -0.10          # 10日跌幅 > 10%
drop_5d > -0.20           # 但不至于踩雷(20%以上可能基本面出问题)
rsi_14 < 30               # 超卖
fund_flow_ratio > 0       # 当日资金净流入为正(承接信号)
vol_reversal > 1.5        # 放量止跌
recovery_signal = 1       # 收阳线
```

### 2.3 评分（软排序）

```
score = -1.0 * rank(drop_10d)           # 跌幅越深越好
      + -0.5 * rank(rsi_14)             # 越超卖越好
      + +1.0 * rank(fund_flow_ratio)     # 资金流入越多越好
      + +0.5 * rank(vol_reversal)        # 放量越大越好
```
