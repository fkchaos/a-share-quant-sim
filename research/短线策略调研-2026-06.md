## 一、淘股吧/雪球 最新短线策略框架

### 1.1 情绪周期+N型主升战法（淘股吧最火，6月热帖）

**核心逻辑**：放弃首波试错，专做有辨识度个股的二次/多次发酵

**走势结构**：3/4/5+N+N 经典行情结构
- 第一波：3+连板完成辨识度打底
- 题材冷却后震荡洗盘
- **反包首板**=二波启动起点（盈亏比最高买点）
- **1进2确定性**=次优买点
- 龙头异动滞涨→切换同赛道低位补涨
- 龙头跌停→无条件离场

**中位出清信号**：
- 6板+龙头断板 → 场内3板以上个股3日内全部断板
- 全部出清次日 → 市场情绪回暖

**拐点确认**：涨停家数持续走高 + 跌停减少 + 封板率提升

**来源**：淘股吧「风一样的胖刺猬」6月21日帖（4423浏览/73评论/14加油）

### 1.2 龙头战法30条（雪球高赞帖，IC可验证）

| 战法 | 入场条件 | 量价验证 | 止损 |
|------|---------|---------|------|
| 龙头首次分歧 | 放量T字板回封 | 换手>15%，回封量≥前日80% | 破10日线+缩量 |
| 弱转强 | 前日烂板，次日高开3%+ | 竞价量>前日10%，高开5%+ | 封单减少→核按钮 |
| 首阴反包 | 首日放量阴但未爆量 | 换手<25%，反包日量>首阴 | 反包量<15%换手 |
| 冰点转折 | 沪深成交<7000亿 | 首板量比>5 | 新题材次日无跟风 |
| 爆量分歧→一致 | 涨停换手>20% | 次日竞价量>前日15%+高开3% | 封单不足→炸板 |

**换手率阈值字典**：
- <3% 不活跃
- 3% 温和活跃
- 7% 比较活跃
- 10% 很活跃（游资票）
- 15% 出货嫌疑
- 20%+ 要么波澜壮阔大行情，要么出货

**来源**：雪球「邓墩朝」2025-07-17帖，30种A股短线战法详细解析

### 1.3 对标型龙头战法（淘股吧A短线女王）

**核心**：龙头走出N板后不追龙头，挖掘同题材低位补涨股打首板

**量化参数**：
- 龙头高位缩量（换手<15%）→ 补涨股放量首板（换手>10%）
- 典型补涨链：宝鼎科技 → 金安国纪 → 华正新材 → 诺德股份

### 1.4 涨停板分层套利（淘股吧6月实盘复盘）

**三层结构**：
1. **中军**（大市值/机构票）：稳指数、托板块、分歧日承接
2. **150亿左右小票趋势情绪龙**：主攻手，弹性最大，但连续上涨必触发异动
3. **补涨标的**：情绪龙断板后，低位同产业链轮动套利

## 三、可量化落地的因子思路汇总

### 3.1 连板辨识度因子

**逻辑**：曾经3+连板的股"记忆效应"强，二次启动概率高
**计算**：过去N日内连板最大次数 + 距离天数衰减
**数据**：daily_kline可检测涨停（收盘价=涨停价），跨日连续涨停可统计
**可行性**：⭐⭐⭐ 直接用现有数据

### 3.2 情绪周期择时因子

**逻辑**：涨停家数/跌停家数/封板率→判断市场情绪阶段
**计算**：
  - 全市场涨停数（排除ST/一字板）
  - 跌停数
  - 封板率 = 封住涨停数/曾触及涨停数
  - 情绪温度计 = 涨停数×封板率 - 跌停数
**可行性**：⭐⭐⭐ 可从stock_pool+daily_kline计算

### 3.3 首阴反包信号

**逻辑**：断板后首日反包=短线做多信号
**计算**：N日内曾涨停→昨日收盘低于涨停价→今日收盘=涨停价
**可行性**：⭐⭐⭐ 日频可做

### 3.4 竞价弱转强

**逻辑**：竞价从-2%拉至+3%+竞价量>前日20%
**数据**：需分钟级/竞价数据（腾讯行情接口可获取）
**可行性**：⭐⭐ 需新增数据源

### 3.5 中位出清拐点

**逻辑**：6板+龙头断板→3板以上3日内全断→次日回暖
**计算**：实时连板梯队统计 + 断板检测
**可行性**：⭐⭐⭐ 连板可从日频数据算

### 3.6 对标补涨因子

**逻辑**：龙头滞涨→同板块低位股放量
**计算**：板块分类 + 涨停股识别 + 板块内低位放量股
**可行性**：⭐⭐ 需板块分类映射

### 3.7 聪明钱因子（券商研报）

**逻辑**：大单成交占比×方向判断机构/游资参与
**数据**：需大单/小单成交数据（Level-2或腾讯接口字段57流通市值推算）
**可行性**：⭐⭐ 需大单数据或用成交量替代

### 3.8 集合竞价特征（打板策略核心）

**逻辑**：竞价量/前日比、竞价涨幅、涨停试盘
**数据**：9:25竞价数据
**可行性**：⭐ 需分钟级数据或实时接口

### 3.9 ML/DL方向

**LSTM动态因子权重**、**深度学习非线性因子**、**强化学习仓位管理**、**图网络行业关联**
**可行性**：⭐ 需大量工程投入+GPU

## 六、补充调研：三大待实验方向深度分析

### 6.1 行业轮动（先选行业再选股）

#### 6.1.1 行业轮动策略核心逻辑

**经济周期理论**：扩张期科技/消费领涨，收缩期医疗/公用事业防御。
**相对强度理论**：表现强势的行业有动量延续性，短期继续强势。

**头部量化实践（中金2026年展望）**：
- 中金构建"市场集中度指数"（公募基金前十大重仓股集中度）
- **共识行情**（集中度高）→ 认知型Alpha（深度研究主线龙头）
- **分歧行情**（集中度低）→ 交易型Alpha（广度分散捕捉轮动）
- **2026年判断**：市场集中度进入"中枢抬升的平台期"，即"复杂主线行情"
- 量化策略需同时具备"深度"（理解主线）+"广度"（捕捉轮动）

#### 6.1.2 行业动量因子实现方案

**经典方案**（BigQuant社区策略）：
```
行业得分 = 0.4 × 近5日行业收益率 + 0.3 × 近20日 + 0.3 × 近60日
```
- 按申万一级行业分类
- 选Top 5行业，行业内等权选股
- 调仓周期：周频或日频

**A股特色行业轮动规律**：
- 2026年成长+小盘风格占优，但估值处于历史高位（80%+分位）
- 高拥挤+高趋势赛道（如有色金属）波动风险大
- 低拥挤+高趋势赛道（电子/机械/化工/建材）性价比更高
- 行业轮动速度在"分歧行情"中加快，2026年3月因行业轮动极端反转导致量化超额收益骤降

#### 6.1.3 与现有策略的融合方式

| 融合方式 | 描述 | 工程复杂度 |
|---------|------|-----------|
| A: 行业过滤 | 选股前先过滤到Top N行业，再在行业内选股 | 低 |
| B: 行业加权 | 因子打分时加入行业动量得分 | 中 |
| C: 独立行业轮动策略 | 单独跑一个行业轮动策略，与v39i/v44并行 | 中 |
| D: 行业轮动择时 | 根据行业轮动速度动态调整仓位/集中度 | 高 |

**推荐路径**：A → B → C，逐步验证

#### 6.1.4 数据需求

- 申万一级行业分类映射（股票→行业）
- 行业指数日频收益率（计算行业动量）
- 行业轮动速度指标（行业排名变化率）
- **数据源**：akshare（爱可分享）可免费获取申万行业分类和指数行情

### 6.2 ML因子（非线性因子组合）

#### 6.2.1 ML选股的标签构造方法论

**标签构造三要素**（来自"土法炼钢"小组）：

1. **预测目标**：截面 t 上股票在持仓窗口 [t, t+h] 内的**排名收益**（Rank Return）
   - 不用绝对收益，用截面排名（消除市场涨跌影响）
   - Rank转换 + 反正态变换：`x = Φ⁻¹((rank - 0.5) / N)`

2. **标签窗口选择**：
   - 短线策略：h = 3~5天
   - 中线策略：h = 10~21天
   - 长线策略：h = 60~252天

3. **防过拟合**：
   - 样本量要求：GBDT需要10万~100万独立样本
   - A股截面5000只 × 10年 ≈ 1200万原始样本
   - 但"等价独立样本"仅 1/20~1/50（截面相关+时间相关）
   - 需做 Purged K-Fold 交叉验证（训练集和测试集之间加 embargo 期）

#### 6.2.2 特征工程要点

**因子预处理**：
- 缺失值填充：截面中位数
- 去极值：MAD法（中位数绝对偏差，3倍MAD截尾）
- 标准化：截面z-score
- 行业中性化：在截面内减去行业均值（消除行业暴露）

**常用因子集**：
| 因子类别 | 具体因子 | 来源 |
|---------|---------|------|
| 价量 | 动量(5/21/60)、换手率、量比、振幅 | 现有数据可算 |
| 质量 | ROE、毛利率、资产负债率 | 财报数据 |
| 成长 | 营收增速、净利润增速 | 财报数据 |
| 分析师 | 盈利预测调整、评级变化 | 需额外数据 |
| 资金流 | 主力净流入、北向资金 | 需额外数据 |
| 波动 | 历史波动率、下行波动率 | 现有数据可算 |
| 技术 | MACD、RSI、布林带 | 现有数据可算 |

#### 6.2.3 模型选择与工程路径

**推荐模型**：LightGBM（梯度提升决策树）
- 训练快、效果好、可解释（SHAP归因）
- 原生支持类别特征
- 对超参数不太敏感

**工程路径**：
```
Phase 1: 单因子ML（用ML给现有因子赋动态权重）
Phase 2: 多因子ML（用ML学习因子间的非线性组合）
Phase 3: 集成学习（多个ML模型投票/Stacking）
```

**关键防坑**：
- ❌ 不能直接用原始价格做特征（非平稳）
- ❌ 不能在全量数据上训练（前视偏差）
- ❌ 不能用高复杂度模型在小样本上训练
- ✅ 必须做 Walk-Forward 验证（与现有WF框架一致）
- ✅ 必须做 SHAP 归因（解释模型在做什么）

#### 6.2.4 与现有框架的整合

现有 `wf_runner.py` 已支持 Walk-Forward 验证，ML因子可以直接复用：
1. 新增 `scripts/strategies/v46_ml_factor.py`
2. 在 `strategy_map.py` 注册
3. 用 `wf_runner.py` 做 WF 验证
4. 对比 v39i/v44 的结果

**工程依赖**：
```bash
pip install lightgbm shap scikit-learn
```

### 6.3 事件驱动（财报/增减持）

#### 6.3.1 A股事件类型与超额收益特征

**中金事件驱动研究（2026年）**：

| 事件类型 | 超额收益窗口 | 持有期 | 年化超额 | 数据获取难度 |
|---------|------------|--------|---------|------------|
| 业绩预告（预增） | 公告后1-5日 | 1-4周 | 5-15% | 低（东方财富/同花顺） |
| 业绩快报（净利润+30%） | 公告后1-3日 | 40交易日 | 8-12% | 低 |
| 大股东增持 | 公告后1-2周 | 1年 | 10-20% | 中 |
| 股权激励实施 | 公告后1-3月 | 1年 | 15-20% | 中 |
| 高送转 | 预案前1-2月 | 至预案日 | 8% | 中 |
| 指数成分股调整 | 公告后1-5日 | 1-2周 | 3-5% | 低 |
| 并购重组 | 公告后1-5日 | 事件驱动 | 不确定 | 高 |

**PEAD（财报后漂移）效应**：
- SUE（标准化盈余意外）Q10组（最高超预期）公告后60日继续漂移+1%~+2%
- Q1组（最低超预期）反向漂移-1%~-2%
- 中间分组无显著漂移
- **策略**：买入SUE Q10组，做空Q1组（多空组合）

#### 6.3.2 事件驱动实现方案

**数据源**：
- 东方财富/同花顺公告数据（网页抓取或API）
- Tushare Pro（需积分，付费可解锁更多数据）
- AKShare（免费，覆盖大部分A股事件）

**实现路径**：
```
Phase 1: 业绩预告事件（数据易获取，逻辑清晰）
Phase 2: 增减持事件（需区分高管/个人 vs 机构）
Phase 3: 多事件复合（事件信号叠加，提高胜率）
```

**与现有策略的融合**：
- 事件信号作为**额外过滤条件**：选股结果中，有正面事件信号的加分
- 或作为**独立策略**：单独跑事件驱动，与v39i/v44并行运行

#### 6.3.3 注意事项

- 事件驱动策略的**资金容量大**（事件多、分散）
- 但**调仓频率低**（事件发生时才交易，非每日调仓）
- 与v39i（每日调仓）的**策略相关性低**，适合作为卫星策略
- 需要注意**信息泄漏**：公告日期 vs 实际可得数据日期要严格区分

## 八、风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| 行业轮动在共识行情失效 | 2026年市场集中度平台期，可能偏共识 | 行业过滤不要太激进，保留30%+股票 |

| 炸板率过高（31%） | 买入后炸板亏损 | 加入封板率/封单量过滤 |
| 策略同质化 | 超额收益衰减 | 多策略并行，低相关性组合 |

## 九、总结

**一句话**：v39i（动量+质量）和v44（资金流+动量+低波）已验证有效，下一步最优操作是**加行业轮动过滤**（P0，低风险快赢），同时**布局ML因子**（P2，中期核心差异化）。

**核心判断**：
- 2026年"复杂主线行情"下，单一因子方向容易失效
- 多因子+多策略并行是头部量化的共识方向
- 差距主要在**因子多样性**和**市场状态识别**
- 先做P0/P1验证，再投入P2/P3，控制试错成本
