# v61b 优化设计方案

## 背景

v61 策略（换手率+小市值）WF 验证通过，夏普 2.3-2.6，正fold 93-100%。
现在尝试进一步优化，提升收益或降低风险。

## v61 当前参数

```python
DEFAULT_PARAMS = {
    'MAX_HOLDINGS': 5,
    'REBALANCE_DAYS': 5,
    'STOP_LOSS': -0.10,
    'TAKE_PROFIT': 0.20,
    'HOLD_DAYS_MAX': 5,
}
```

因子: 换手率5日均值(负向) + 市值(负向)，等权50/50

## 优化方向

### 方案A: 调整因子权重
- 测试不同换手率/市值权重比例
- 例如: 70/30, 60/40, 40/60, 30/70

### 方案B: 调整调仓周期
- 测试 3天、7天、10天 调仓
- 观察对收益和换手率的影响

### 方案C: 调整持仓集中度
- 测试 3只、8只 持仓
- 观察对收益和风险的影响

### 方案D: 添加新因子
- 添加动量因子（与v39g形成互补）
- 添加波动率因子（低波筛选）

## 实验计划

1. 先用组合测试脚本快速筛选最优参数
2. 对最优参数跑完整WF验证
3. 与v61对比，选择更优版本

## 验证标准

- 夏普 > 2.0
- 正fold > 90%
- 与v61对比有明显提升
