## 一、当前策略

### v12_multi — 多策略并行（全量回测最优，WF 未通过）

| 维度 | 值 |
|------|-----|
| **选股池** | 中证800（715只） |
| **评分模式** | Multi-Strategy（多策略并行加权） |
| **子策略** | v11b(ensemble, w=0.5) + v10c(factor, w=0.3) + v6b_hlr(factor, w=0.2) |
| **调仓频率** | 20 交易日 |
| **止损/止盈** | 20% / 3档 |
| **持有期衰减** | ✅ |
| **行业限制** | 25% |
| **波动率缩放** | ✅（target=20%） |

**原理**：多个异构策略独立评分 → 各自 z-score 标准化 → 按权重加权混合。
不同策略的评分量纲不同，必须标准化后再混合。

**回测结果（close, 2021-01~2026-06, 中证800）：**

| 指标 | v12_multi | v11b | v10c | v6b_hlr |
|------|----------|------|------|---------|
| 年化 | **32.77%** | 26.25% | 37.33% | 22.64% |
| Sharpe | **1.14** | 1.05 | 1.30 | 1.23 |
| 最大回撤 | **-24.19%** | -25.97% | -27.86% | -19.63% |

**vs v11b 单策略**：年化 +6.5pp，回撤 -1.8pp

**子策略角色：**
| 子策略 | 模式 | 权重 | 角色 |
|--------|------|------|------|
| v11b_zz800_union | ensemble | 0.5 | 主力（WF 验证最优） |
| v10c_zz800_balanced | factor | 0.3 | 补充（中证800 IC 最优因子） |
| v6b_hlr | factor | 0.2 | 稳定器（低回撤基准） |

**Walk-Forward（16 folds, 252d训练/63d测试）：**

|------|-----------------|----------|----------|------|

| 平均夏普 | 0.19 | **0.74** | **0.80** | **1.70** |
| 正收益 fold | 8/16 (50%) | 8/16 (50%) | 7/16 (44%) | **11/16 (69%)** |

**权重扫描（5 种组合）：**

| 权重 (v11b/v10c/v6b) | 年化 | 夏普 | 正收益 |
|----------------------|------|------|--------|

**结论**：❌ WF 未通过。所有权重组合正收益 fold < 60%。
多策略混合在样本外不稳定，z-score 对齐噪声大。v11b 单策略依然最优。
**状态**：归档，不作为模拟盘候选。

**v20c 评分函数：**
```
score = 缩量加分(0~3.0) + 收窄加分(0~2.0) + 活跃加分(0~1.0) + 趋势加分(0~1.0) + 股性加分(0.8)
```
- 缩量加分：vol_ratio 0→3.0分, 0.5→2.0, 1.0→1.0, 1.5→0（线性）
- 收窄加分：range_ratio 0→2.0分, 0.5→1.0, 1.0→0（线性）
- 活跃加分：amount_ratio=1.0→1.0分，偏离越多分越低（高斯形状）
- 趋势加分：price_vs_ma5>1.0→最高1.0分
- 股性加分：20天内有涨停→0.8分
- 硬性排除：vol_ratio>1.5×上限 或 amount_ratio<0.3×下限

**设计逻辑：**
1. 尾盘缩量 + 振幅收窄 = 洗盘结束信号
2. 次日虽然可能低开，但日内反弹强劲
3. 价格 > MA5 保证短期趋势向上
4. 涨停历史保证股性活跃

## 七、2026-07-14 新策略测试结论

### v27 价量共振（WF 通过 ✅，已上线账户2）

| 维度 | 值 |
|------|-----|
| **选股池** | 中证800（715只，排除科创板/北交所） |
| **评分模式** | 动量 + 量价共振 + 辅助因子 |
| **核心因子** | mom_5（基础分）+ pv_corr_20（价量共振）+ gap + illiq + boll |
| **排除条件** | pv_corr_10 < -0.5（量价严重背离）+ delist_risk 前10% |
| **调仓频率** | 每日调仓 |
| **持仓周期** | 2-5天（hold_days_max=5） |
| **止损/止盈** | -2% / +5% |
| **最大持仓** | 8只 |
| **单只上限** | 20% |
| **初始资金** | 10万（账户2） |
| **市场状态控制** | ✅ 牛市1.0/震荡0.7/熊市0.3（影响可用资金比例） |
| **模拟盘脚本** | `scripts/sim/account_runner.py --strategy v27` |

**回测结果（2022-2026）：**

| 指标 | 全量 | WF |
|------|------|-----|
| 年化 | 251% | 48.8%（fold平均） |
| Sharpe | 5.72 | 8.66 |
| 最大回撤 | -6.7% | -1.74% |
| 正收益 fold | — | 15/15 (100%) |

**vs v22（纯动量）：**
- 全量年化持平（251% vs 252%）
- 全量夏普更高（5.72 vs 5.54）
- 全量回撤更小（-6.7% vs -7.1%）
- WF 结果远优于 v22（v22 未跑 WF）

**vs v13（反转+量价）：**
- v27 是动量方向（mom_5 > 2%），v13 是反转方向（rev_5 < -2%）
- v27 WF 夏普 8.66 vs v13 WF 夏普 3.43
- 两者选股重叠度极低（<5%），高度互补

**设计逻辑：**
1. mom_5 > 2% 作为基础门槛
2. pv_corr_20 > 0 表示量价同步（放量上涨/缩量下跌 = 健康趋势）
3. 排除 pv_corr_10 < -0.5 的量价严重背离股票
4. gap/illiq/boll 作为辅助加分项
5. 退市风险硬过滤

**IC 分析关键发现：**
- pv_corr_20 IC_IR = 0.135（有效因子）
- pv_corr_10 IC_IR = 0.106（短期也有效）
- illiquidity IC_IR = 0.275（最强单因子）
- gap_ratio IC_IR = 0.162（第二强）

**参数扫描结论：**
- mom_threshold 在 0.005~0.05 范围内结果几乎无影响
- 所有阈值选出 Top 8 股票基本相同
- 建议保持默认 0.02

## 九、架构重构记录（2026-07-13）

### 9.1 账户-策略解耦

**问题**：每个策略对应一个独立的 sim 脚本，切换策略 = 切换脚本，无法灵活组合。

**方案**：
- 新增 `core/strategy_map.py`：策略名称 → 选股函数的映射表
- 新增 `scripts/sim/account_runner.py`：统一模拟盘入口（`--strategy v27|v20c`）
- 选股逻辑独立到 `scripts/strategies/`，可被回测和模拟盘共用

**结果**：新增策略只需写选股函数 + 注册映射表，不改账户脚本。

### 9.2 目录重构

`scripts/` 从平铺 100+ 文件改为分层结构：

| 目录 | 内容 |
|------|------|
| `scripts/sim/` | 模拟盘执行层（account_runner + legacy 脚本） |
| `scripts/strategies/` | 选股逻辑模块（v27_select, v20_tail_pick） |

| `scripts/tools/` | 工具脚本（数据更新、导入等） |
| `scripts/archive/` | 归档旧版本 |

### 9.3 account_runner load/save 修复

**问题**：原代码调用不存在的 `load_account_for_sim` / `save_account_for_sim`。

- `load_account(id)`：`get_account(id)` 读现金 + `get_holdings(id)` 读持仓，从 `added_at` 计算 `hold_days`
- `save_account(state, id)`：`upsert_account` 写现金 + `upsert_holding` 写持仓 + `delete_holding` 删已清仓
