## Week 5 (2026-06-22(一) ~ 2026-06-25(四))

### 策略
- **v39 系列迭代（v39c → v39d → ... → v39i）**：
  - **v39d IC 驱动权重调整**：IC 分析发现 size_factor(IC=+0.049/ICIR=+0.30) 为唯一有效因子，mom_5 IC 为负；权重重构：W_MOM 0.20→0.15, W_SIZE 0.10→**0.30**, W_ILLIQ 0.10→0.20, 其余因子降权；风控全面放宽：SL -1.5%→**-5%**, TP 3%→**10%**, BUY 4→3
  - **v39e/v39f/v39g 风控微调全部失败**：收紧止损(-3%)被震出(+31.32%/0.534)、降低止盈(5%)无效(+30.14%/0.442)、缩短持有期(3天)无效(+20.48%/0.377)
  - **v39h/v39i 动态 MOM_THRESHOLD ✅ 重大突破**：熊市自动提高选股门槛自然减仓；5×5=25组参数扫描，最优 BULL=0.05/BEAR=0.08
  - **v39i 最终参数**：W_MOM=0.15, W_SIZE=0.30, W_ILLIQ=0.20, W_PV_CORR=0.05, W_TURNOVER=0.05, W_FUND_FLOW=0.05, W_GAP=0.05；SL=-5%, TP=10%, BUY=3
  - **v39i 全量回测（2023-2025）**：**+103.51%** / 夏普 **1.199** / 回撤 **16.69%**（超越 v39d 的 +35.33%/0.456）；2023年熊市从亏6%→赚22%（熊市自动减仓）
  - 已上线账户2（替换 v27）

- **v40 因子恶化卖出**：持仓重评分 < SELL_THRESHOLD → 卖出；证伪——硬风控（SL/HOLD_MAX）已先于因子恶化行动，框架可用但无增量价值
- **v40b 纯轮动**：每日卖最低4只+买最高4只，无硬风控；WF 15 folds 平均 -24.03%/夏普 -1.963/1/15 正收益 ❌；根因：日换手100% → 年化交易成本121%

- **v41 VWAP 偏离+净支撑量因子**：新增 2 个量价因子；所有新因子 IC 为负，因子组合已达最优，不需要加新因子

- **v42 换手率因子研究**：真实换手率 vs 量比；clip_max 无影响，换手率与动量 r=0.94 高度冗余；结论：换手率因子不自成体系

- **v43 全A追涨策略证伪**：小市值轮动追涨，全量回测 **-15.78%** / 夏普 **-2.322**；全A 4933只 OOM 问题暴露

- **v44 资金流+动量+低波质量 ⭐ 新最优夏普策略**：
  - W_FLOW=0.35（核心），W_MOM=0.25，W_SIZE=0.15，低波过滤（日vol≤4%），动量门槛>3%，zz800 范围
  - **WF（4 folds）**：**+26.32%** / 夏普 **1.252** / 回撤 **18.71%** / 正收益 fold **4/4 (100%)** ✅
  - 夏普 1.252 超越 v39i（1.199），风险调整后收益最优
  - 13种权重组合 + 11种风控组合结果几乎一致，策略鲁棒性强

- **v45 新方向研究启动**：反转策略（超跌放量反弹）、短周期调仓（v39i 变体 HOLD_MAX=3）、事件驱动（财报/增减持/解禁）、真·行业轮动（先选行业再选个股）

### 架构
- **load_panel() 重写为 load_panel_from_db()**：批量 SQL 替代逐只 get_kline，~3s 替代 480s+，性能提升 160x
- **股票池配置化**：pool 字段在 strategy_map.py 中统一管理，默认 zz800，全A策略需显式 pool='full_a'
- **cmd.py 数据库 CLI**：新增 status/switch/signals/trades/kline 等子命令，交互式数据库管理
- **archive/ 目录创建**：归档过时脚本和实验代码，主工程结构更清晰
- **文档清理**：移除 cli.py 引用，全篇策略版本更新（v27→v39i→v44），cron prompt 精简
- **GitLab remote 已删除**：项目不再依赖 GitLab

### 踩坑记录
- **A 股中止损太紧比太松更危险**：-1.5% 止损在 A 股震荡行情中被反复震出（2023年 -29.66%），放宽到 -5% 后扭亏为正
- **参数修改必须同步 strategy_map.py 和策略文件**：v39d→v39e 时漏改一处参数导致结果偏差半天
- **因子权重调整必须先做 IC 分析**：v39c 权重分配与 IC 不匹配是最大问题（W_MOM=0.20 但 IC 为负）
- **文档必须面向普通用户**：不硬编码特定环境路径，使用相对路径和环境变量
- **全A回测 OOM**：4933 只股票 × 3 年数据超出内存，需缩短回测窗口或缩减选股范围
