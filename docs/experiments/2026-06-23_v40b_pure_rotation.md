# v40b 实验记录 — 每日无条件轮动

## 实验目的
验证"每日无条件轮动"策略在 A 股市场的有效性：
- 每天卖出持仓中评分最低的 4 只
- 每天买入全市场评分最高的 4 只
- 无硬风控（止损/止盈/持有期限制）

## 策略设计

### 核心逻辑
```python
# 每天执行：
sell_list = 持仓中评分最低的 4 只
buy_list  = 全市场评分最高的 4 只（排除已持有）
```

### 参数
- `SELL_COUNT = 4`
- `BUY_COUNT = 4`
- `NO_HARD_RISK = True`
- 评分权重与 v39c 完全一致（sum=1.0）

### 代码文件
- `scripts/strategies/v40_factor_exit.py` — `select_stocks_v40b()` 函数
- `core/strategy_map.py` — v40b 注册（NO_HARD_RISK=True）
- `core/strategy_adapter.py` — v40b 注册到 _register_builtin_strategies
- `scripts/backtest/wf_runner.py` — v40b 轮动分支

## 失败原因分析

### 1. 高换手率带来的交易成本
- 每天换手率接近 100%（卖4买4 = 8次交易/天）
- 每次交易成本：佣金 0.03% + 滑点 0.03% = 0.06%
- 每天交易成本：8 × 0.06% = 0.48%/天
- 年化交易成本：0.48% × 252 = **121%/年**

### 2. 追涨杀跌效应
- 评分最高的股票往往是近期强势股（动量因子权重 0.20）
- 第二天容易回调，买入后立即亏损
- 卖出评分最低的股票时，这些股票可能已经超卖反弹

### 3. 无复利效应
- 无法持有优质股超过 1 天
- 没有"让利润奔跑"的机制
- 每次买入都是全新仓位，无法积累收益

### 4. 与 v39c 对比
- v39c 持仓期平均 5 天，换手率约 40%/月
- v40b 持仓期 1 天，换手率 100%/天
- v39c 夏普 1.18 vs v40b 夏普 -1.963

## 教训

1. **高换手率是 A 股量化的大敌**：交易成本远超过选股优势
2. **追涨杀跌在 A 股无效**：A 股动量效应弱，反转效应强
3. **需要降低换手频率**：如每 3-5 天轮动一次
4. **或加入卖出过滤条件**：如只卖亏损+低分的股票
5. **无条件轮动不如有条件持有**：v39c 的硬风控虽然简单，但有效控制了换手率

## Bug 修复记录

### wf_runner.py 买入循环位置错误
- **问题**：买入循环在 `elif` 块内部，v40b 的 `if` 分支满足时跳过了买入循环
- **修复**：将买入循环从 `elif` 块中提出，放到 `if/elif` 之后对所有策略统一执行
- **影响**：v40b 之前所有 fold 收益率为 0%（未发生任何交易）
