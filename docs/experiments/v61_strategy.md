# v61 换手率+小市值 低流动性溢价策略 — 设计文档

> 创建: 2026-06-29
> 状态: WF 验证通过 ✅

## 背景

BigQuant 上"换手率3因子低流动性溢价策略"年化 38.24%、夏普 1.16。
在 zz1800 内复现后发现：换手率单因子效果一般，但**换手率+小市值组合**效果极佳。

## 策略逻辑

**核心思想：买最不活跃的小票 = 低流动性溢价 + 小市值溢价**

### 因子定义

| 因子 | 公式 | 方向 | 权重 |
|------|------|------|------|
| 换手率5日均值 | mean(volume/float_shares, 5) | 负向（低=好） | 50% |
| 市值 | close × float_shares | 负向（小=好） | 50% |

### 选股规则

1. 每 5 天调仓一次
2. 两个因子分别 rank（0-1 分位），等权相加
3. 选综合得分最高的 5 只
4. 等权持仓

### 风控

- 手续费：买万三 + 卖千一
- 无止损（纯因子驱动）

## 回测结果

### 全量回测（2021-01-04 ~ 2026-06-26）

| 指标 | 值 |
|------|-----|
| 总收益 | +1733% |
| 年化 | +73.81% |
| 夏普 | 1.636 |
| 最大回撤 | -39.38% |
| 日胜率 | 58.9% |
| 交易次数 | 594 |

### WF 验证（16 folds, train=252, test=126, step=63）

| 指标 | 值 |
|------|-----|
| 平均收益/fold | +38.96% |
| 平均夏普 | 2.342 |
| 平均回撤 | -17.7% |
| 正收益 fold | 15/16 (93.75%) |
| **状态** | **✅ PASS** |

### 对比 v39g

| 指标 | v61 | v39g |
|------|-----|------|
| 年化 | +73.81% | +16.02% |
| 夏普 | 1.636 | 1.164 |
| 回撤 | -39.38% | -14.0% |
| 正fold | 15/16 | 12/16 |

## 风险提示

1. **全量回测收益偏高**（+1733%），WF 更可信（+38.96%/fold）
2. **低换手率股票流动性差**，实际滑点会比回测大
3. **小票波动大**，回撤 -39.38% 高于 v39g 的 -14.0%
4. **与 v39g 低相关**，可组合使用

## 复现来源

BigQuant: https://bigquant.com/square/ai/5c58cce8-fba7-e78e-c579-0c7340a07d2b
原版：3 个换手率因子，年化 38.24%，夏普 1.16
我们：2 个因子（换手率+市值），年化 73.81%，夏普 1.636

# v61b 优化设计方案

```python
DEFAULT_PARAMS = {
    'MAX_HOLDINGS': 5,
    'REBALANCE_DAYS': 5,
    'STOP_LOSS': -0.10,
    'TAKE_PROFIT': 0.20,
    'HOLD_DAYS_MAX': 5,

```

因子: 换手率5日均值(负向) + 市值(负向)，等权50/50

- 测试不同换手率/市值权重比例
- 例如: 70/30, 60/40, 40/60, 30/70

- 测试 3天、7天、10天 调仓
- 观察对收益和换手率的影响

### 方案C: 调整持仓集中度
- 测试 3只、8只 持仓
- 观察对收益和风险的影响

- 添加动量因子（与v39g形成互补）

# v61b Barra 风格归因分析设计文档

> 日期: 2026-06-30
> 分析目标: v61b 策略（换手率+小市值）的收益归因 — Alpha vs 风格暴露

---

## 背景

- v61b 是当前部署在账户1的最优策略（WF 夏普 2.300）
- 核心因子: 换手率5日均值(负向) + 市值(负向)，等权 ranking，卖出即买
- 旧 Barra 归因脚本（v11b）基于 CSV 数据源 + zz800 股票池，需全面重写

## 分析问题

v61b 的收益中：
1. 有多少来自 **风格暴露**（低换手率溢价、小市值溢价）？
2. 有多少来自 **Alpha**（选股能力，无法被风格解释的超额）？
3. 策略赚的是什么钱？因子组合是否真正提供了独立 Alpha？

## 方法论

### 风格因子构建（Barra 标准）

| 因子 | 定义 | 数据来源 |
|------|------|---------|
| SIZE | log(流通市值) | close × volume 估算（无 float_shares 表） |
| MOM_5/10/20 | 过去 N 日收益率 | close.pct_change(N) |
| VOL_5/20 | 过去 N 日收益标准差 | returns.rolling(N).std() |
| LIQ | log(过去20日日均成交额) | amount.rolling(20).mean() |
| REV_5 | 过去5日反转（负收益） | -close.pct_change(5) |
| BETA | 市场 Beta（60日滚动） | cov(r_i, r_m) / var(r_m) |
| TURNOVER | 换手率5日均值 | amount / (close × 1亿) 估算 |

### 归因方法

**Step 1: 横截面 IC 分析**
- 计算每个风格因子与未来5日收益的 Spearman IC
|IC Mean| > 0.03 → 因子有独立预测力

**Step 2: 横截面回归归因**
- 每期截面: R_i = α + β_SIZE×SIZE_i + β_MOM×MOM_i + ... + ε_i
- 看 α 是否显著（t-stat > 2）
- α 显著 → 策略有独立 Alpha（风格无法解释）

**Step 3: 时间序列回归归因**
- 用 v61b 的 WF 回测 NAV，分解策略收益:
  R_strategy(t) = α + β_1×F_SIZE(t) + β_2×F_MOM(t) + ... + ε(t)
- Alpha 占比 = α / 策略总收益
- 风格占比 = Σ(β_i × F_i 年化收益) / 策略总收益

### 数据源

- K线数据: `data/quant_stocks.db` → `daily_kline` 表
- 股票池: `stock_pool_zz1800` 表（1800只）
- 策略收益: 从 `quant_accounts.db` → `trade_log` 重建，或跑 WF 获取 NAV
- 时间范围: 2021-01-01 ~ 2026-06-24（标准回测区间）

### 与 v11b 归因的差异

| 维度 | v11b | v61b |
|------|------|------|
| 股票池 | zz800 (800只) | zz1800 (1586只) |
| 核心因子 | MOM/ILLIQ/SIZE 等 | 换手率 + SIZE |
| 持仓风格 | 大盘股偏动量 | 小市值+低换手 |
| 预期结论 | 纯 Alpha | 可能风格占比更高（小市值+低换手本身就是风格） |

## 产出

- 脚本: `scripts/tools/barra_attribution_v61b.py`
- 报告: `docs/experiments/2026-06-30_v61b_barra_attribution.md`
- 结论: Alpha 占比 / 风格占比 / 策略赚的是什么钱
