## P0 建议（Agent 标为"必须做"）

### P0-1：接入 AKShare / Tushare 作为备用数据源

**评估：⚠️ 需讨论**

- **现实约束**：当前沙箱环境 AKShare 连不通（`push2.eastmoney.com` RemoteDisconnected），腾讯接口是唯一可用源
- **可行方案**：可以在 `update_daily_data.py` 加一个 fallback 层，实现逻辑是"腾讯失败时尝试 Tushare"，但当前环境下无法验证
- **建议**：暂缓。等网络环境变化或迁移到有完整数据源的环境再接入。当前先保证腾讯接口稳定即可

### P0-2：添加基本面因子（PE/PB/ROE等）

**评估：✅ 采纳，低成本高价值**

- **可行性**：腾讯行情接口可以获取部分估值数据；或者用 Tushare/AKShare 定期批量下载基本面数据到 CSV
- **实现成本**：低。只需在 `core/factors.py` 里新增几个因子类（`PEFactor`、`PBFactor`、`ROEFactor`），数据一次下载、本地缓存
- **收益**：从纯技术面升级为技术+基本面多因子模型，策略维度更丰富
- **优先级**：高。作为下一个开发模块

## P1 建议（Agent 标为"应该做"）

### P1-1：Walk-Forward 分析

**评估：✅ 采纳，中等成本高价值**

- **可行性**：在 `run_backtest.py` 中可以增加 `walk_forward()` 函数：将数据按时间切分为训练期/测试期滚动验证
- **实现成本**：中。需要重构回测主循环支持滚动窗口，约 1-2 天
- **收益**：过拟合检测的"金标准"，对评估策略有效性有重大价值
- **优先级**：中。因子优化完之后做

### P1-2：回测曲线可视化

**评估：✅ 采纳，低成本**

- **可行性**：在 `run_backtest.py` 的 `save_results()` 里增加 matplotlib 生成净值曲线图
- **实现成本**：低。几行代码：
  ```python
  import matplotlib.pyplot as plt
  nav.plot(); plt.savefig('nav_curve.png')
  ```
- **收益**：直观展示回测结果，debug 和策略对比必备
- **优先级**：高。几乎零成本，立刻可做

### P1-3：更丰富的绩效指标（Calmar/Sortino）

**评估：✅ 采纳，零成本**

- **现状**：`run_backtest.py` 里已经有 Calmar 比率
- **补充**：Sortino 比率（只计下行波动）加一行代码即可
- **实现成本**：极低
