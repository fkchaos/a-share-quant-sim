# v39h/v39i 动态 MOM_THRESHOLD 实验（2026-06-23）

## 背景

v39d 年度分析发现核心问题：
- 2023年熊市：-6.34%（止损太松+持仓太久）
- 2024年牛市：+29.16%
- 2025年牛市：+30.14%（夏普2.02，极好）
- **结论**：策略本身在牛市非常强（夏普2.02），但熊市拖累整体。

## 尝试的方向

### 1. 风控参数微调（v39e/v39f/v39g）—— 全部失败

| 版本 | 改动 | 收益 | 夏普 | 结论 |
|------|------|------|------|------|
| v39e | SL -5%→-3% | 31.32% | 0.534 | 止损太紧被震出 |
| v39f | TP 10%→5% | 30.14% | 0.442 | 降止盈无效 |
| v39g | HOLD_MAX 5→3 | 20.48% | 0.377 | 短持有期无效 |

### 2. 新因子尝试（v41）—— 失败

- 行业动量、质量因子、VWAP偏离、净支撑量
- **所有新因子 IC 为负！** 只有 size_factor + illiq 有正 IC
- 结论：因子组合已最优，不需要加新因子

### 3. 动态 MOM_THRESHOLD（v39h/v39i）—— ✅ 重大突破

**核心思想**：不修改风控参数，而是让选股门槛随市场状态自动调整。
- 牛市：正常选股门槛（mom_5 > 3%~5%）
- 熊市：提高门槛（mom_5 > 8%~10%），自然减少/不选股，持仓自然清仓

## 实现细节

```python
def _get_mom_threshold(factors, date, params):
    m5 = factors['mom_5'].loc[date].dropna()
    median_mom = m5.median()
    
    if median_mom > 0:
        return params["MOM_THRESHOLD"]       # 牛市门槛
    else:
        return params["MOM_THRESHOLD_BEAR"]  # 熊市门槛
```

**关键参数扫描结果（25种组合）：**

| MOM_BULL | MOM_BEAR | 收益 | 夏普 | 回撤 |
|---------|---------|------|------|------|
| 0.03 | 0.10 | 115.42% | 1.145 | 23.49% |
| **0.05** | **0.08** | **103.51%** | **1.199** | **16.69%** |
| 0.04 | 0.10 | 109.37% | 1.084 | 25.18% |
| 0.05 | 0.10 | 98.10% | 0.970 | 33.05% |

**为什么 BULL=0.05 / BEAR=0.08 最优？**

- 牛市门槛 0.05：比 0.03 更严格，只选最强势的 5%，减少噪音
- 熊市门槛 0.08：比 0.10 更宽松，熊市也能选到少量优质股
- 极端值（0.01/0.15）表现差：太松或太紧都有问题

所有方向均已尝试并收敛：

| 扫描维度 | 范围 | 结果 |
