## 一、活跃策略

### v67 — v66 + 情绪择时（⭐ 当前运行，账户2）

| 维度 | 值 |
|------|-----|
| **选股池** | 中证1800（1586只，排除科创板） |
| **评分模式** | 8因子横截面归一化 + 加权评分 |
| **核心因子** | size_factor, illiq, mom_5, turnover, fund_flow, gap_ratio, pv_corr, two_day_limit |
| **选股门槛** | 动量>3% + pv_corr_10 > -0.5 + delist_risk 排除 + **情绪择时** |
| **情绪择时** | market_sentiment > 阈值时才交易（避免冷清市） |
| **评分权重** | W_MOM=0.08, W_SIZE=0.35, W_ILLIQ=0.16, W_PV_CORR=0.04, W_TURNOVER=0.04, W_FUND_FLOW=0.04, W_GAP=0.04, W_TWO_DAY_LIMIT=0.35 |
| **调仓频率** | 每日调仓 |
| **止损/止盈** | -5% / +5% |
| **持有期** | max 3天，浮盈≥8%延长到3天 |
| **最大持仓** | 5只，每日最多买4只，单只上限20% |

**回测结果（标准WF，15 folds，2021-01-01 ~ 2026-06-24）：**

| 指标 | v67 | v39g（同期） | 提升 |
|------|-----|-------------|------|
| 收益 | +8.59% | +8.59% | 0% |
| Sharpe | **1.724** | 1.189 | **+45%** |
| 最大回撤 | **-6.80%** | -14.53% | **-53%** |
| 正收益fold | 12/15 (80%) | 11/15 (73%) | +7% |

**⚠️ 注意**：v67在近一年（2025-06~2026-05）夏普1.712 < v66夏普2.105，情绪择时在活跃市略有拖累。

**核心创新：情绪择时**
- 计算连续两天涨停股票数的滚动均值作为市场情绪指标
- 阈值=2.0，窗口=15天
- 情绪低时（冷清市）暂停交易，避免在低活跃期亏损

**策略文件：** `scripts/strategies/v67.py`

### v39i — 动态 MOM_THRESHOLD（⚠️ 历史策略）

| 维度 | 值 |
|------|-----|
| **选股池** | 中证800（708只，排除科创板） |
| **评分模式** | 7因子横截面归一化 + 加权评分 |
| **核心因子** | size_factor, illiq, mom_5, turnover, fund_flow, gap_ratio, pv_corr |
| **选股门槛** | **动态 MOM_THRESHOLD**（见下方逻辑说明） + pv_corr_10 > -0.5 + delist_risk 排除 |
| **评分权重** | W_MOM=0.15, W_SIZE=0.30, W_ILLIQ=0.20, W_PV_CORR=0.05, W_TURNOVER=0.05, W_FUND_FLOW=0.05, W_GAP=0.05 |
| **调仓频率** | 每日调仓 |
| **止损/止盈** | -5% / +10% |
| **持有期** | max 5天，浮盈≥3%延长到10天 |
| **最大持仓** | 8只，每日最多买5只，单只上限12.5% |
| **初始资金** | 20万（账户2） |

**回测结果（2023-01-01 ~ 2026-06-27 全量，zz800/708只）：**

| 指标 | v39i | v39g（同期） | 备注 |
|------|------|-------------|------|
| 收益 | **+44.12%** | +10.51% | v39i 胜 |
| Sharpe | **0.513** | 0.239 | v39i 胜 |
| 最大回撤 | 28.26% | 32.40% | v39i 略优 |

**⚠️ 股票池依赖**：v39i 在 zz800（大盘股）表现更优，v39g 在 zz1800（含小市值）表现更优。

**核心创新：动态 MOM_THRESHOLD**
- 每日计算全市场 mom_5 中位数
- 中位数 > 0（牛市）：门槛 = 0.05（正常选股，取最强势的5%）
- 中位数 ≤ 0（熊市）：门槛 = 0.08（大幅提高，自然减仓/空仓）

**策略文件：** `scripts/strategies/v39i_optimized.py`

## 三、待验证策略

### v46a — v39i + 行业动量过滤 + 连板因子 + 业绩预告（❌ 已证伪）

| 维度 | 值 |
|------|-----|
| **基础策略** | v39i（夏普1.199/回撤16.69%/WF 4正） |
| **增强模块** | 行业动量Top10过滤 + 连板因子加分 + 业绩预告过滤 |
| **选股池** | zz1800（1800只） |
| **状态** | ❌ 全市场验证失败（2026-06-26） |

**WF 结果**：

| 窗口 | v46a | v39i（同期） | 差距 |
|------|------|-------------|------|
| 2023-06~2024-06（熊市） | -21.08%/夏普-2.253 | -9.30%/夏普-0.725 | 多亏12pp |
| 2024-06~2025-12（牛市） | -11.53%/夏普-1.001 | +15.87%/夏普1.183 | 多亏27pp |

**失败原因**：行业动量过滤在 A 股短线选股中严重滞后（行业轮动极快），三重过滤叠加导致选股过窄。行业动量因子对 v39i 是**显著负增量**。

**后续**：放弃 v46a 组合方案。连板因子和业绩预告因子可独立重测（不与行业过滤叠加）。

## 四、架构记录

### wf_runner.py 命令行参数

```bash
python scripts/backtest/wf_runner.py --strategy <name> [options]

--strategy   策略名 (必填)
--train      训练期天数 (默认: 252)
--test       测试期天数 (默认: 252)
--step       滑动步长 (默认: 252)
--start      回测起始日期
--end        回测结束日期
--full       全量回测模式（不做 WF 切分）
--pool       覆盖策略股票池 (zz800/zz1800/full_a)
```

**`--pool` 用法**：
```bash
# 用 zz800（708只排除科创板）跑 v39g
python wf_runner.py --strategy v39g --full --pool zz800

# 用 zz1800（1586只排除科创板）跑 v39g
python wf_runner.py --strategy v39g --full --pool zz1800

# 不传 --pool，使用 strategy_map 中配置的 pool（v39g/v39i 默认 zz1800）
python wf_runner.py --strategy v39g --full
```

### 策略注册完整流程

新增策略必须完成以下 4 步：

1. **创建 `scripts/strategies/vXX.py`**：select_stocks_vXX 函数
2. **注册 `core/strategy_map.py`**：select_fn + calc_factors_fn + params
3. **注册 `scripts/backtest/strategy_adapter.py`**：_register 方法 + _risk_params + _vXX_select 方法
4. **注册 `scripts/backtest/wf_runner.py`**：_calc_factors elif 分支

**验证命令：**
```bash
python -c "from scripts.backtest.strategy_adapter import get_adapter; print(get_adapter().list_strategies())"
python -c "from core.strategy_map import list_strategy_names; print(list_strategy_names())"
```
