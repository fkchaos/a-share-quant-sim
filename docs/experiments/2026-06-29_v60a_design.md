# v60a: 行业中性化 — 设计文档

> 创建: 2026-06-30
> 目标: 对 v39g 因子做行业中性化，降低行业集中风险
> 方法: 用 industry_map 表做截面行业均值去化（无需 AlphaPurify 的回归方法）

---

## 1. 逻辑

原始因子 → 按行业分组减去行业均值 → 中性化后的因子

**为什么重要**:
- v39g 重 SIZE 因子 → 选出的小票可能集中在某几个行业
- 行业中性化可降低行业轮动带来的波动
- 已经有 industry_map 表（5456只 → 申万行业）

## 2. 方法

对截面因子值 `f(date, stock)`：
```
neutral_f(date, stock) = f(date, stock) - mean(f(date, industry(stock)))
```

这是最简单的**行业均值中性化**，优于不做，且不需要额外依赖。

## 3. 实验设计

在 v39g 的选股框架上对比：
- A组（基准）: v39g 原始因子
- B组（实验）: v39g 因子行业中性化后选股

同条件: train=252, test=126, step=63, 2021-2025, zz1800
