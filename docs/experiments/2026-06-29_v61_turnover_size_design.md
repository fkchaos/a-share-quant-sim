# v61 换手率+小市值 低流动性溢价策略 — 设计文档

> 创建: 2026-06-29
> 状态: WF 验证通过 ✅

## 背景

BigQuant 上"换手率3因子低流动性溢价策略"年化 38.24%、夏普 1.16。
在 zz1800 内复现后发现：换手率单因子效果一般，但**换手率+小市值组合**效果极佳。

## 策略逻辑

**核心思想：买最不活跃的小票 = 低流动性溢价 + 小市值溢价**

### 因子定义

| 因子 | 公式 | 方向 | 权重 |
|------|------|------|------|
| 换手率5日均值 | mean(volume/float_shares, 5) | 负向（低=好） | 50% |
| 市值 | close × float_shares | 负向（小=好） | 50% |

### 选股规则

1. 每 5 天调仓一次
2. 两个因子分别 rank（0-1 分位），等权相加
3. 选综合得分最高的 5 只
4. 等权持仓

### 风控

- 手续费：买万三 + 卖千一
- 无止损（纯因子驱动）

## 风险提示

1. **全量回测收益偏高**（+1733%），WF 更可信（+38.96%/fold）
2. **低换手率股票流动性差**，实际滑点会比回测大
3. **小票波动大**，回撤 -39.38% 高于 v39g 的 -14.0%
4. **与 v39g 低相关**，可组合使用
