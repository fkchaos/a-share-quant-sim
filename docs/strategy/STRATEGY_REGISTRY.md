## 一、活跃策略

### v39i — 动态 MOM_THRESHOLD（⭐ 当前最优，运行账户2）

| 维度 | 值 |
|------|-----|
| **选股池** | 中证800（707只，排除科创板/北交所） |
| **评分模式** | 7因子横截面归一化 + 加权评分 |
| **核心因子** | size_factor(最强), illiq, mom_5, turnover, fund_flow, gap_ratio, pv_corr |
| **选股门槛** | **动态 MOM_THRESHOLD**（见下方逻辑说明） + pv_corr_10 > -0.5 + delist_risk 排除 |
| **评分权重** | W_MOM=0.15, W_SIZE=0.30, W_ILLIQ=0.20, W_PV_CORR=0.05, W_TURNOVER=0.05, W_FUND_FLOW=0.05, W_GAP=0.05 |
| **调仓频率** | 每日调仓 |
| **止损/止盈** | -5% / +10% |
| **持有期** | max 5天，浮盈≥3%延长到10天 |
| **最大持仓** | 8只，每日最多买3只，单只上限20% |
| **初始资金** | 20万（账户2） |

**回测结果（2023-2025, 全量727天）：**

| 指标 | v39i | v39d（旧版） | 改进 |
|------|------|-------------|------|
| 收益 | **+103.51%** | +35.33% | **+193%** |
| Sharpe | **1.199** | 0.456 | **+163%** |
| 最大回撤 | **16.69%** | 47.37% | **-65%** |
| 最终净值 | **407,026** | 270,653 | **+50%** |

**年度分解：**

| 年份 | 收益 | 夏普 | 回撤 |
|------|------|------|------|
| 2023（熊市） | **+21.79%** | 1.09 | 15.75% |
| 2024（牛市） | **+75.26%** | 1.58 | 28.33% |
| 2025（震荡） | **+18.71%** | 1.00 | 11.11% |

**核心创新：动态 MOM_THRESHOLD**
- 每日计算全市场 mom_5 中位数
- 中位数 > 0（牛市）：门槛 = 0.05（正常选股，取最强势的5%）
- 中位数 ≤ 0（熊市）：门槛 = 0.08（大幅提高，自然减仓/空仓）
- **效果**：2023年熊市从 v39d 的-6.34%变成 +21.79%

**策略文件：** `scripts/strategies/v39i_optimized.py`
**注册文件：** `core/strategy_map.py`（v39i）, `scripts/backtest/strategy_adapter.py`（_v39i_select）

### v44 — 质量+低波小市值（⭐ 研究中）

| 维度 | 值 |
|------|-----|
| **选股逻辑** | 全A 4605只，中小盘20-100亿，质量(20日动量)+低波动(20日波动率)+规模+换手率质量 |
| **评分权重** | W_QUALITY=0.35, W_LOWVOL=0.30, W_SIZE=0.20, W_TURNOVER=0.15 |
| **选股门槛** | 动量>0% + 日波动率≤5% + 换手率≥0.2% |
| **止损/止盈** | -8% / +15% |
| **持有期** | max 10天，浮盈≥3%延长到15天 |
| **调仓频率** | 每日调仓 |

**设计理念**：2024年低波因子表现优秀，2025年质量+规模因子共振。2026年风格不明，质量+低波组合攻守兼备。
**参考**：guorn.com 2025年A股因子分析（规模/质量/成长领涨，低波/价值输家）

### v44 — 资金流+动量+低波质量（⭐ 新最优，研究中）

| 维度 | 值 |
|------|-----|
| **选股逻辑** | zz800范围，资金流(0.35)+动量(0.25)+规模(0.15)+换手率(0.10)+gap(0.05)+illiq(0.10) |
| **选股门槛** | 动量>3% + 日波动率≤4% |
| **止损/止盈** | -5% / +10% |
| **持有期** | max 5天，浮盈≥3%延长到12天 |
| **调仓频率** | 每日调仓 |

**WF结果（2023-2026, 4 folds）**：**+26.32% / 夏普1.252 / 回撤18.71% / 正收益fold 4/4 (100%)** ✅

**参数稳定性**：v44 对风控参数（止损/止盈/持有期）极不敏感——所有组合（SL=-3%/-5%/-8%, TP=8%/10%/15%/20%, Hold=5/10/15天）结果均为 +19.58%~+26.32%/夏普0.93~1.25。说明收益来自选股逻辑本身而非风控调优。

**设计理念**：资金流=动量×成交量变化（价量共振强度），在v39d中fund_flow权重0.15已验证有效。v44将其提升到0.35作为核心，并加入低波动率过滤（日vol≤4%）提升质量。

## 三、架构记录

### 策略注册完整流程

新增策略必须完成以下 4 步：

1. **创建 `scripts/strategies/vXX.py`**：select_stocks_vXX 函数
2. **注册 `core/strategy_map.py`**：select_fn + calc_factors_fn + params
3. **注册 `scripts/backtest/strategy_adapter.py`**：_register 方法 + _risk_params + _vXX_select 方法
4. **注册 `scripts/backtest/wf_runner.py`**：_calc_factors elif 分支

**验证命令：**
```bash
python -c "from scripts.backtest.strategy_adapter import get_adapter; print(get_adapter().list_strategies())"
python -c "from core.strategy_map import list_strategy_names; print(list_strategy_names())"
```

### 账户-策略分离

- 账户在 DB `account.strategy` 字段绑定策略
- 策略不绑定账户，可随时切换
- 统一入口：`account_runner.py run intraday_signal --account-id <id>`
