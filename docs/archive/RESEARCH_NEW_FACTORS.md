## 问题分析

1. **多重共线性**：新因子间高度相关（如 illiquidity 和 amplitude 都基于价格波动）
2. **权重分配**：v6b 60% + 新因子 40% 是拍脑袋的，没有优化
3. **过拟合风险**：illiquidity IC_IR=0.31 异常高，可能是数据窥探

## 下一步建议

3. **权重优化**：用 IC_IR 加权而非固定比例

## 附录：国泰君安 Alpha 191/144 因子调研（2026-06-07）

### Alpha 191 体系

- **来源**：国泰君安金工刘富兵团队，2017年6月《基于短周期价量特征的多因子选股体系》
- **数量**：191个纯价量因子（OHLCV + VWAP）
- **预测周期**：T+1 至 T+4（短周期）
- **研测结果**：2012-2017年，中证500基准，年化超额50.2%，IR=4.67，最大回撤5.9%
- **容量**：1-7亿元（市场依赖）
- **换手率**：~50倍（双边）

### Alpha 144：下跌日平均波动

- **描述**（来源 BigQuant 因子列表）："下跌日平均波动"
- **类别**：波动率与风险类（Alpha 121-150区间）
- **逻辑**：衡量下跌日（CLOSE < OPEN）的平均日内波动幅度，与总体平均波动对比
- **精确公式**：未找到原始公式。建议回测时自行实现：
  ```python
  # 伪代码
  down_days = (close < open)
  down_vol = (high - low).where(down_days).rolling(20).mean()
  all_vol = (high - low).rolling(20).mean()
  alpha_144 = down_vol / all_vol
  ```
- **交易含义**：高值=下跌日波动大（恐慌分歧），低值=下跌平稳

### 结论

1. **不建议直接采用 Alpha 191**：已发布9年，多数因子可能衰减；容量太小不适合大资金
2. **可用作参考框架**：Alpha 因子的构造思路（价量背离、波动率异常等）可以借鉴
3. **如果要使用**：建议只选 IC/IR 验证后仍有效的子集，作为 v13 评分排序的补充因子
4. **需要自己做 IC 验证**：用 `scripts/ic_analysis_zz800.py` 在本地数据上验证 Alpha 144 因子的 IC/IR
