## 核心结论

**策略以 Alpha 为主，风格暴露占比极小。**

| 来源 | 年化收益 | 占比 |
|------|---------|------|
| 策略总收益 | 24.51% | 100% |
| Alpha (纯选股) | 25.03% | **102.1%** |
| 风格因子贡献 | 0.04% | **0.2%** |

## 关键发现

### 1. Alpha 主导
- Alpha 在 10% 水平下边际显著 (p=0.0728)
- 风格因子回归 R² = 0.0119，仅解释策略 1.2% 的收益波动
- **策略收益几乎全部来自选股能力，不是风格暴露**

### 2. 风格暴露方向
| 因子 | 暴露方向 | 年化贡献 | 显著性 |
|------|---------|---------|--------|
| VOL_60 | 负向（低波动偏好） | -0.07% | 弱 |
| BETA | 正向（高Beta偏好） | +0.05% | 不显著 |
| VOL_20 | 正向 | +0.04% | 不显著 |
| SIZE | 正向（偏大盘） | +0.01% | 不显著 |
| LIQ | 正向（偏流动性好的） | +0.01% | 不显著 |

### 3. IC 分析
| 因子 | IC_IR | 显著性 |
|------|-------|--------|
| LIQ | -0.403 | ** |
| SIZE | -0.317 | ** |
| MOM_60 | -0.218 | * |

**LIQ 和 SIZE 的 IC 最显著**，说明：
- 流动性差的股票未来收益更低（负 IC）
- 小市值股票未来收益更低（负 IC，与我们选股池偏大盘一致）

### 4. 截面回归
- 平均 R² = 0.2313，截面回归解释力中等
- 平均 Alpha = 0.010116（每期调仓日）
- 仅 LIQ 因子在 10% 水平显著

## 对策略的启示

### 好消息
1. **Alpha 来源清晰**：策略赚的是选股钱，不是风格暴露的钱
2. **风格中性**：没有过度依赖某一风格因子，策略稳健性较好
3. **低波动偏好**：VOL_60 负向暴露说明策略天然偏好低波动股票，这是好的风险调整

### 可改进方向
1. **Alpha 显著性不够强** (p=0.0728)：纯选股能力有，但不够突出
2. **风格因子解释力低** (R²=1.2%)：说明还有大量 Alpha 来源未被模型捕捉
3. **可考虑加入风格因子择时**：如果能在不同市场状态下调整风格暴露，可能提升 Sharpe

## 下一步建议
1. ✅ 当前策略 Alpha 主导，风格中性 — 不需要大改
2. 可尝试在 Ensemble 中加入风格因子作为额外一组
3. 关注 Alpha 的稳定性 — 如果 Alpha 衰减，需要新因子来源
